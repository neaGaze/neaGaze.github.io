Portfolio

Claimbuster

It is a tool that helps fact checkers find a factual claim that are worth checking. My roles in this project was in the development of Slack bot using Node.JS and classifying factual claims into different lexicographical models using Python

SportsHub

An android app that helps the Sports organizer connect with the players in a university sporting events using the Node.JS backend server. My responsibility was the development of both the mobile and backend part.

CrowdConsensus

It is a Slack bot developed using Node.JS and MongoDB that helps in collaborative group decision making using statistical and probabilistic techniques, and inputs derived from crowdsourcing

BigMart

A e-commerce portal for a retailing corporation. My key roles were the development of features such as the ViewPagers slides, UI animations, handling webservice calls and deployment.

Period Tracker

An android app that keep track of the periods, records the moods, weights, temperature, fertility periods and more. I was responsible for the development of entire app along with testing and deployment.
